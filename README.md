# 习惯养成认知神经模拟项目
基于 Hebb 学习规则与突触可塑性原理 | 全自动多习惯并行模拟

## 项目核心思路
本项目基于 Hebb 学习规则与突触可塑性原理模拟人类习惯养成过程：为神经元设置初始微小权重，以 0/1 行为触发信号驱动突触权重动态更新；引入时间计时器记录行为间隔，根据间隔长短调整权重强化 / 衰减系数；最终通过神经元权重总和量化习惯强度。

**升级功能：**
- 全自动行为模拟（无需人工点击）
- 多组习惯并行对比（不同频率）
- 时间等比加速（快速观察长期效果）
- 实时可视化习惯强度变化曲线
- 计算神经科学标准模拟逻辑

## 核心技术
- Hebb 学习规则（突触可塑性）
- 时间依赖的强化与遗忘机制
- 全自动多习惯并行模拟
- 时间尺度缩放（加速实验）
- Streamlit 交互式可视化界面
- Python + NumPy 科学计算
